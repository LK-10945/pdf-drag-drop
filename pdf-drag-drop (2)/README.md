# PDF Metadata Manager

A React-based app to upload PDF files with metadata, built using:

- Vite
- Tailwind CSS
- Framer Motion
- React Dropzone

## Features

- Drag and drop PDF upload
- Animated drag state
- Metadata form (type + notes)
- Simulated progress bar
- Responsive and clean UI
